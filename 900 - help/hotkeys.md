# Hotkeys (Obsidian)

Source of truth: `.obsidian/hotkeys.json`.

Notes:
- **Mod** = `Ctrl` (Windows/Linux) / `Cmd` (macOS).
- Some hotkeys may reference plugins you used historically; if a plugin is currently removed/disabled, that hotkey will do nothing until the plugin is re-enabled.

## Core / Workspace

| Action (command id) | Hotkey |
|---|---|
| Toggle left sidebar (`app:toggle-left-sidebar`) | Mod+Alt+V |
| Toggle right sidebar (`app:toggle-right-sidebar`) | Mod+Alt+B |
| Toggle pin (`workspace:toggle-pin`) | Mod+Alt+P |
| Split vertical (`workspace:split-vertical`) | Mod+Alt+R |
| Open current tab in new window (`workspace:open-in-new-window`) | Mod+Alt+O |
| Toggle source / live preview (`editor:toggle-source`) | Mod+Alt+E |
| Fold all (`editor:fold-all`) | Mod+Alt+M |
| Unfold all (`editor:unfold-all`) | Mod+M |
| Open local graph (`graph:open-local`) | Mod+Alt+G |
