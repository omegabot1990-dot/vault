- An Event is <mark style="background: #BBFABBA6;">a specific outcome or a set of outcomes</mark> from a [[202602061354 - random experiment|random experiment]]
- While an _outcome_ is a single result, an _event_ can be one result, multiple results, or even no results at all
- An event is a subset of the [[202602061346 - sample space|Sample Space]] ($S$), $E \subseteq S$
- Types of Events
	1. <mark style="background: #FFF3A3A6;">Simple Event</mark>
		1. Contains only one outcome (e.g., rolling exactly a $3$)
	2. <mark style="background: #ABF7F7A6;">Compound Event</mark>
		1. Contains multiple outcomes (e.g., rolling a number $> 4$, which is $\{5, 6\}$)
	3. <mark style="background: #FF5582A6;">Impossible Event</mark>
		1. A set with no outcomes (e.g., rolling a $7$ on a 6-sided die), $P(E) = 0$
	4. <mark style="background: #D2B3FFA6;">Certain Event</mark>
		1. A set containing the entire sample space, $P(E) = 1$
