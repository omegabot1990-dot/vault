- A Random Variable is a function that maps the [[202602061346 - sample space|sample space]] $S$ (all possible outcomes) to the set of real numbers $\mathbb{R}$ [^1]

> [!MATH] Random Variable ($X$)
> $$X: S \to \mathbb{R}$$

- Types of Random Variables
	1. <mark style="background: #BBFABBA6;">Discrete</mark>
		1. Outcomes you can count (e.g., number of students in a class, result of a die roll)
	2. <mark style="background: #BBFABBA6;">Continuous</mark>
		1. Outcomes that can be any value in a range (e.g., exact height, time remaining, temperature)
