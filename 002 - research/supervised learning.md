## Topics

- Supervised learning uses labeled examples $(x,y)$ to learn a mapping from inputs to targets
- The objective is to minimize prediction error on unseen data from the same distribution
- Main task types are classification (discrete labels) and regression (continuous targets)
- Typical pipeline includes train/validation/test split, loss minimization, and generalization monitoring
