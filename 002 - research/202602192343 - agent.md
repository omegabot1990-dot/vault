- An agent is the <mark style="background: #BBFABBA6;">decision-making entity</mark> that interacts with an [[202602192345 - environment|environment]] over time
- At each step, it observes [[202602192351 - state|state]] information, chooses an [[202602192350 - action|action]], and receives a [[202602192352 - reward|reward]] signal
- The agent aims to maximise [[202602061333 - expectation|expected]] cumulative [[202602192355 - return|return]] through its [[202602192245 - policy|policy]]
- Agent behaviour is updated from experience using [[202602192359 - value estimation|value estimation]], [[202602200001 - policy optimization|policy optimization]], or both
- The quality of an agent depends on:
	- <mark style="background: #D2B3FFA6;">Representation</mark>
	- <mark style="background: #FFF3A3A6;">Exploration strategy</mark>
	- <mark style="background: #ABF7F7A6;">Learning algorithm</mark>
- <mark style="background: #FF5582A6;">Agent-environment interaction defines the core feedback loop of RL</mark>
