- [[202602010044 - model|Model]] parameters are <mark style="background: #BBFABBA6;">values learned from data</mark> during [[202602111335 - training|training]]
- They are updated by optimisation steps to reduce the [[202602010047 - objective function|loss]]
- In neural networks, weights and biases are model parameters
- Model parameters differ from [[202602111256 - hyperparameters|hyperparameters]], which are set before training
- Parameter count controls model capacity and memory/computation cost
- Learned parameter values determine the final function used at inference
