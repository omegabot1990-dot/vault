- Return is the cumulative future [[202602192352 - reward|reward]] from a time step onward
- Return defines the optimization target for [[202602192245 - policy|policies]] and [[202602200020 - value|value]] functions
- Discounting down weights distant rewards by a factor $\gamma\in[0,1)$

- [[202602192356 - episode|Episodic]] return from time $t$:

> [!MATH] Episodic Return
  >$$G_t=\sum_{k=0}^{T-t-1}\gamma^k R_{t+k+1}$$

- Infinite-horizon discounted return:

> [!MATH] Infinite-Horizon Discounted Return
> $$G_t=\sum_{k=0}^{\infty}\gamma^k R_{t+k+1}$$

- <mark style="background: #BBFABBA6;">Higher expected return indicates better long-term decision quality under the task objective</mark>
