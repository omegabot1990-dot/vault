#### [Pandas](https://pandas.pydata.org/docs/user_guide/10min.html) (Jupyter)

* It is a Python library used for data manipulation
* It is well-suited for structured data
	* Tables of statistics
	* Time-series data
* Key Features
	* Data Structures:
		* DataFrame is for tabular data
		* Series is a single column of a DataFrame
	* Data Manipulation:
		* Provides many data manipulation functionalities akin to Excel
		* Can process data faster than an Excel
	* Reading and Writing data
		* CSV
		* Excel
		* SQL database
		* JSON
		* etc.
	* Handling Missing Data
	* Merging and Joining
	* Grouping and Aggregating
