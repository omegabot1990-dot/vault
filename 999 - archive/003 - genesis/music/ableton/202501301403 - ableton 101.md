## Defaults

1. Default of -12db set for Audio and MIDI tracks to create ==headroom==

## Automation

1. Automation Button
	1. `opt` will make the slope curve when held down
2. Automation Lock Button
	1. If turned on, it will lock the drawn automation 

## Connecting Guitar

> Scarlett Solo

1. Guitar is added to the Audio Track
2. Live Settings -> Audio -> Input and Output device
3. Set a lower buffer size in Live Settings -> Audio for latency issues
	1. 128
	2. 64
4. Devices
	1. Tuner

## MIDI Keyboard

> Launch Key Mini MK3

1. Connect LaunchKey Mini to Ableton
	1. [Unofficial Video](https://www.youtube.com/watch?v=-3fkAsimPIs&t=692s)
		1. Very informative
		2. ==Watch again==
	2. [Official Playlist](https://www.youtube.com/@NovationHelp/playlists)
2. Live Settings -> Link, Tempo and MIDI -> Takeover Mode
	1. Set to Pickup Mode
3. MIDI Mapping
