## Keyboard Shortcuts

|       Action        |      Keyboard Shortcut      |
|:-------------------:|:---------------------------:|
| Insert Audio Track  |           `cmd T`           |
|  Insert MIDI Track  |        `shift cmd T`        |
| Insert Return Track |         `opt cmd T`         |
|       Rename        |           `cmd R`           |
|      Duplicate      |           `cmd D`           |
|      Quantise       |        `cmd shift U`        |
|     Group Items     |           `cmd G`           |
|      Draw Mode      |             `B`             |
|   Automation Mode   |             `A`             |
|   Loop Selection    |           `cmd L`           |
|     Consolidate     |           `cmd J`           |
|  View -> Help View  |         `opt cmd 7`         |
|   Octave Up/Down    | `shift up arrow/down arrow` |
- Double Click on buttons to bring it back to the **default** values
